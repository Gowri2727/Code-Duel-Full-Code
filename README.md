# ⚔️ Code-Duel

AI-powered real-time coding battle platform where users compete by solving coding problems and win based on code efficiency.

---

## 🚀 Features

- AI-generated coding questions
- Random matchmaking
- Friend room creation
- Multiplayer coding battles
- Ready system
- Countdown timer
- Python support
- Winner based on efficiency
- View opponent code

## 🛠 Tech Stack

- React.js

- Node.js
- Express.js
- MongoDB

**AI**
- OpenRouter AI

## ⚙️ Run Project

Frontend:

```bash
cd client
npm install
npm run dev
```

Backend:

```bash
cd server
npm install
npm start
```
